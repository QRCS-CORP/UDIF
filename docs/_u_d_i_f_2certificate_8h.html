Detailed Description

UDIF certificate operations.

This module implements certificate generation, signing, verification, and serialization for the UDIF trust hierarchy. Certificates bind entities to their public keys and define their capabilities.

Certificate Hierarchy:

  • Root: Self-signed, starts the trust chain
  • Branch Controllers: Signed by Root or parent Branch
  • Group Controllers: Signed by Branch
  • User Agents: Signed by Group Controller
