Detailed Description

UDIF console help functions.

Two functions mirror the MPDC pattern:

udif_help_print_context — prints one help line for a specific action. The action enum value is used as an index into UDIF_APPLICATION_HELP_STRINGS in resources.h.

udif_help_print_mode — prints all help lines relevant to the current console mode and entity role. The role parameter allows the certificate and server sub-modes to show role-specific commands (e.g. "sign" only for Root).

Print all help lines for the current console mode and role.

Iterates over the commands that are valid in the given mode, printing each one with udif_help_print_context. For certificate and server sub-modes, only commands applicable to the given role are printed.

Parameters
prompt[const] The current command prompt string.
modeThe current console mode.
roleThe entity role (governs which sub-mode commands appear).
